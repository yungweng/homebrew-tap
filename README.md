# homebrew-tap

Homebrew formulas for tools by [yungweng](https://github.com/yungweng).

## Usage

```bash
brew tap yungweng/tap
brew install pr-codex-review
```

## Formulas

- [pr-codex-review](https://github.com/yungweng/pr-codex-review): run parallel Codex PR reviews and aggregate them into one GitHub comment.
